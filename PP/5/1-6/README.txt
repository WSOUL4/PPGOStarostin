person
Это структура с полями имя и возраст
PrintPerson
Функция для вывода информации о человеке
Birthday
Функция для увеличения возраста человека на 1

В основном модуле мы создаём алису 23 лет выводит её данные, а затем используем указатель для ускорения функции дня рождения и снова выводим информацию


Shape
Интерфейс который имеет в себе метод "Площадь" 
AreaI
Это функция, которая заставляет объект пользующийся интерфейсом вызвать собственную "Площадь"

func (c Circle) Area()
Это определение функции интерфейса для определённого объекта, который им пользуется
AreaForAll 
использует типичный ... для работы со всеми вхождениями интерфейса и заставляет его вызвать свой метод площадь, который потом вызывает свой для каждого объекта
